# Internship-Task2
 Sales & Profit Analysis Dashboard (Power BI)

 Overview
This project presents a Power BI dashboard analyzing sales and profitability across categories and regions. It provides insights into revenue trends, profit performance, and business opportunities.

 Dataset
The dataset includes:
- Order Date
- Product Name
- Category
- Region
- Quantity
- Sales
- Profit

 Key Visuals
- KPI Cards (Total Sales, Profit, Orders, Quantity)
- Sales & Profit Trend
- Sales by Category
- Profit by Category
- Sales by Region
- Sales vs Profit Analysis

 Key Insights
- Sales show overall growth trend.
- Some categories generate high revenue but lower profit margins.
- Regional performance varies significantly.
- Higher sales do not always mean higher profitability.

 Tools Used
- Power BI
- DAX
- CSV Dataset
